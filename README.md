Este projeto é um simples SPA criado para estudo.

Utiliza 
<br> 
HTML | SASS | JAVASCRIPT | GULP
<br>

*Para ver em funcionamento basta clonar o repositorio.* 

Instalar todas as dependências:
```
npm i
```

Rodar o projeto em localhost:
```
gulp
```
